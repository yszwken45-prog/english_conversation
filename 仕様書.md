## 1. アプリ概要

| 項目 | 内容 |
|---|---|
| アプリ名 | 生成AI英会話アプリ |
| フレームワーク | Streamlit |

## 2. 画面構成

### 2-1. ログイン画面（未ログイン時）
- ログインタブ：ユーザー名 + パスワードで認証
- 新規登録タブ：ユーザー名（2文字以上）・パスワード（6文字以上）で登録

### 2-2. サイドバー（ログイン済み）
- ユーザー名の表示
- ナビゲーションボタン（学習 / ダッシュボード / 設定 / 管理者※）
- ログアウトボタン
- 学習履歴（日付選択 → 過去の会話を表示）
- バージョン番号表示

※ 管理者ボタンはユーザー名が `admin` の場合のみ表示

### 2-3. 学習ページ（メイン）
学習モードを選択して英会話を練習する画面。

| 操作 | 内容 |
|---|---|
| 開始ボタン | 選択中のモードを開始 |
| 再生速度 | 2.0 / 1.5 / 1.2 / 1.0 / 0.8 / 0.6 倍速から選択 |
| モード選択 | 日常英会話 / シャドーイング / ディクテーション |
| 英語レベル | 初級者 / 中級者 / 上級者（設定ページの保存値がデフォルト） |

### 2-4. ダッシュボードページ
ユーザーの学習実績を可視化する画面。

| 表示内容 | 説明 |
|---|---|
| 総会話数 | ユーザーの発言回数の累計 |
| 学習日数 | 学習を行ったユニークな日数 |
| 最多使用モード | 最も使われた学習モード |
| 直近30日の学習活動 | 日別の会話数を棒グラフで表示 |
| モード別利用状況 | モードごとの会話数を棒グラフで表示 |

### 2-5. 設定ページ
ユーザーごとのAIパーソナライズ設定を保存する画面。

| 設定項目 | 選択肢 | 説明 |
|---|---|---|
| AIの話し方 | フレンドリー / フォーマル / ビジネス | 日常英会話モードのAIトーンに反映 |
| デフォルト英語レベル | 初級者 / 中級者 / 上級者 | 学習ページのレベル選択の初期値 |

設定は「設定を保存」ボタンでDBに保存され、次回ログイン時から反映。

### 2-6. 管理者ページ
ユーザー名 `admin` のみアクセス可能。

| 表示内容 | 説明 |
|---|---|
| 総ユーザー数 | 登録済みユーザーの件数 |
| 総メッセージ数 | 全ユーザーの累計メッセージ数 |
| ユーザー一覧テーブル | ユーザー名・登録日時・総会話数・学習日数 |

## 3. 学習モード仕様

### MODE_1：日常英会話
1. 「開始」ボタンを押す
2. マイクで発話（5秒の沈黙で録音終了）
3. Whisper（whisper-1）で文字起こし
4. GPT-4o-mini（ConversationChain）が返答を生成
5. TTS（tts-1 / alloy）で音声再生
6. 選択した再生速度で出力

AIトーンはユーザー設定（フレンドリー / フォーマル / ビジネス）がシステムプロンプトに付加される。

### MODE_2：シャドーイング
1. AIが約15語の英文を生成してTTSで読み上げ
2. ユーザーが音声でシャドーイング
3. Whisperで文字起こし
4. GPTが元の英文と比較して発音・再現度を評価

### MODE_3：ディクテーション
1. AIが約15語の英文を生成してTTSで読み上げ
2. ユーザーが聞こえた英文をチャット欄に入力して送信
3. GPTが元の英文と入力を比較してスペル・文法を評価

## 4. データベース設計

**ファイル:** `app_data.db`（SQLite、gitignore対象）

### テーブル: `users`
| カラム | 型 | 説明 |
|---|---|---|
| id | INTEGER PK | ユーザーID |
| username | TEXT UNIQUE | ユーザー名 |
| password_hash | TEXT | PBKDF2ハッシュ |
| salt | TEXT | ソルト |
| created_at | TIMESTAMP | 登録日時 |

### テーブル: `conversation_history`
| カラム | 型 | 説明 |
|---|---|---|
| id | INTEGER PK | レコードID |
| user_id | INTEGER FK | ユーザーID |
| session_date | TEXT | 学習日（YYYY-MM-DD） |
| role | TEXT | `user` / `assistant` / `other` |
| content | TEXT | メッセージ内容 |
| mode | TEXT | 使用モード |
| created_at | TIMESTAMP | 記録日時 |

### テーブル: `user_settings`
| カラム | 型 | デフォルト | 説明 |
|---|---|---|---|
| user_id | INTEGER PK FK | — | ユーザーID |
| ai_tone | TEXT | フレンドリー | AIの話し方 |
| english_level | TEXT | 初級者 | デフォルト英語レベル |

## 5. ファイル構成

```
.
├── main.py           # メインアプリ（UI・制御フロー・全ページの描画）
├── functions.py      # ステートフルなヘルパー（音声・TTS・LangChainチェーン）
├── constants.py      # 定数・システムプロンプト・バージョン番号
├── database.py       # SQLite操作（init・CRUD・集計クエリ）
├── auth.py           # PBKDF2認証
├── requirements.txt  # 依存パッケージ
├── .env              # OPENAI_API_KEY（gitignore対象）
├── app_data.db       # SQLiteデータベース（gitignore対象）
├── images/
│   ├── ai_icon.jpg
│   └── user_icon.jpg
├── audio/
│   ├── input/        # 録音ファイル（一時）
│   └── output/       # TTS出力ファイル（一時）
└── logs/
    └── app.log       # RotatingFileHandler（WARNING以上）
```

## 6. 主要な依存ライブラリ

| ライブラリ | バージョン | 用途 |
|---|---|---|
| streamlit | 最新 | UIフレームワーク |
| openai | 最新 | Whisper / TTS / GPT-4o-mini |
| langchain | 0.1.20（固定） | ConversationChain / Memory |
| langchain-openai | 0.1.6（固定） | ChatOpenAI |

## 7. バージョン管理ルール

### セマンティックバージョニング（`メジャー.マイナー.パッチ`）

| 変更の種類 | 例 | 上げる桁 |
|---|---|---|
| 大規模な機能追加・破壊的変更 | 認証方式の変更 | `1.0.0 → 2.0.0` |
| 機能追加 | 新しい学習モード | `1.0.0 → 1.1.0` |
| バグ修正・軽微な改善 | UI崩れ修正 | `1.0.0 → 1.0.1` |

### リリース手順

1. `constants.py` の `VERSION` を更新
2. コミット＆タグ付け

```bash
git add .
git commit -m "リリース内容の説明"
git tag -a v1.1.0 -m "v1.1.0 - ○○機能を追加"
git push origin main --tags
```
